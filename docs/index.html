隱私權政策

本擴充功能「NTU Course Search Helper」不會收集、儲存或傳輸任何個人資料。

收集的資料

  • 本擴充功能不收集任何使用者資料。

資料使用方式

  • 本擴充功能只會在瀏覽器中讀取課程網頁內的文字，用來組合搜尋關鍵字。
  • 搜尋關鍵字由使用者發起，並在使用者的瀏覽器中開啟 Google 搜尋頁面。

資料分享

  • 本擴充功能不會將任何資料傳送至開發者或第三方伺服器。

安全性

  • 本擴充功能不使用遠端程式碼、不注入第三方腳本。
  • 僅在指定的課程頁面運作。
